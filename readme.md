# Learning ExpressJS
### Install NodeJS
***
### Npm Init
***
##### **Npm Init** to create the package.json file. It contains all the dependencies used in the project.
<br>

### Install express
***
#### Use the command. Node modules and package-lock.json are created in the folder.
> npm install express
<br>

### Uninstall express
***
#### Use the command
> npm uninstall express
